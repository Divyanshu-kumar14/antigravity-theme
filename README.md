# Antigravity Theme

Antigravity is a sleek, modern, and high-tech "deep space" aesthetic theme for Omarchy Linux with Hyprland.
It bridges deep black/gray backgrounds with vibrant neon cyan and purple accents.

# Installation Theme

To install this theme, use the `omarchy-theme-install` command and point it to the repository OR use your local path:

```bash
omarchy-theme-install /home/rtx/Desktop/antigravity-theme
```

Or when hosted online:
```bash
omarchy-theme-install <repository-url>
```

## Superfile Support
- This theme includes a Superfile theme config. 
- Copy `antigravity.toml` to `~/.config/superfile/theme/` and edit your `config.toml` to set `theme = 'antigravity'`.
